
## Description
Nestjs framework to solve DS-Algo in API format

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
## Path for mac, if nest is not recognised after global install

export PATH=$PATH:/Users/yourname/.npm-global/bin
